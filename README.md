# Netflix Data Analysis

## Overview
This project performs exploratory data analysis (EDA) on Netflix movie data from the 1990s.

The analysis answers two main questions:

1. What was the most frequent movie duration in the 1990s?
2. How many short action movies (less than 90 minutes) were released in the 1990s?

---

## Tools Used
- Python
- pandas
- matplotlib

---

## Dataset
Netflix dataset containing movie and TV show information.
